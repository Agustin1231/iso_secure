02 / Cambios recientes
// Changelog v2.0 — Actualizaciones implementadas
  • empresa_id agregado a incidents, controls y risk_levels — filtrado automático por empresa según rol
  • Roles admin/auditor ven todos los datos; supervisor/analista solo ven datos de su empresa
  • Controles ISO 27001:2022 auto-asignados al crear/actualizar empresa según actividad económica (A.5, A.6, A.7, A.8)
  • Endpoint POST /empresas/{id}/generar-controles para regenerar controles de empresas existentes
  • Videos reales de YouTube embed en los 4 cursos de capacitación
  • Sidebar corregido en desktop (position:fixed + grid-column:2) y mobile (overflow-y:auto)
  • Migración automática en deploy via migrate.py — idempotente y sin pérdida de datos

04 / Endpoints de la API
/incidents
5 endpoints
GET /incidents Listar todos los incidentes con filtros
POST /incidents Reportar nuevo incidente
GET /incidents/{id} Obtener detalle específico
PUT /incidents/{id}/status Actualizar estado
GET /incidents/stats KPIs de incidentes
/controls
4 endpoints
GET /controls Listar controles ISO 27001
POST /controls Registrar nuevo control
PUT /controls/{id} Actualizar cumplimiento
GET /controls/compliance % global de cumplimiento
/risk
4 endpoints
GET /risk/current Nivel de riesgo actual
POST /risk Registrar evaluación
GET /risk/history Historial por período
GET /risk/by-domain Riesgo por dominio ISO
/dashboard
3 endpoints
GET /dashboard/summary Resumen ejecutivo
GET /dashboard/kpis Todos los KPIs
GET /dashboard/export Export JSON
/empresas
6 endpoints
GET /empresas Listar empresas
GET /empresas/{id} Detalle empresa
POST /empresas Registrar + generar controles
PUT /empresas/{id} Actualizar empresa
POST /empresas/{id}/generar-controles Regenerar controles ISO
DEL /empresas/{id} Eliminar empresa
/auth
4 endpoints
GET /auth/me Perfil usuario actual
GET /auth/users Listar usuarios (admin)
PUT /auth/users/{id}/role Asignar rol + empresa
POST /auth/register-profile Crear perfil inicial
/capacitaciones
6 endpoints
GET /capacitaciones Listar cursos con progreso
GET /capacitaciones/{id} Detalle + progreso usuario
POST /capacitaciones Crear curso (admin)
PUT /capacitaciones/{id} Editar curso (admin)
DEL /capacitaciones/{id} Eliminar curso (admin)
PATCH /capacitaciones/{id}/progreso Actualizar progreso propio
/implementacion
1 endpoint
GET /implementacion/{empresa_id} Dominios ISO recomendados por actividad
/auditoria
6 endpoints
GET /auditoria/empresa/{id} Items de auditoría por empresa
GET /auditoria/empresa/{id}/resumen Resumen progreso auditoría
POST /auditoria/empresa/{id}/generar Generar items desde controles
POST /auditoria Crear item manual
PUT /auditoria/{id} Actualizar estado item
DEL /auditoria/{id} Eliminar item
