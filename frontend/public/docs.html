Descripción del Proyecto

ISO_SECURE es una API REST desarrollada con FastAPI que expone endpoints para consultar, registrar y analizar los Indicadores Clave de Rendimiento (KPIs) del Sistema de Gestión de Seguridad de la Información (SGSI) de una organización, alineados con la norma ISO/IEC 27001:2022 y el marco de medición ISO/IEC 27004.

El sistema permite a la alta dirección disponer de información cuantitativa y trazable sobre el estado de seguridad de la información, facilitando la toma de decisiones, la mejora continua bajo el ciclo PHVA (Planificar-Hacer-Verificar-Actuar) y el cumplimiento normativo.

Alineación con ISO/IEC 27001:2022

El diseño de la API responde directamente a la Cláusula 9 de ISO 27001 (Evaluación del Desempeño), la cual exige que la organización evalúe el rendimiento y la eficacia del SGSI de manera sistemática.

ISO 27001 — Cláusula 9.1
Los endpoints /incidents/stats y /controls/compliance proveen las métricas de desempeño requeridas.
ISO 27001 — Cláusula 9.3
/dashboard/summary genera el reporte ejecutivo consolidado para la alta dirección.
ISO 27005 — Gestión de riesgos
El módulo /risk implementa la matriz probabilidad × impacto (score 1-25) por dominio.
